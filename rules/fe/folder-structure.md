# Folder Structure Rules

## Organization

- Group by feature when possible
- Keep shared logic isolated
- Separate UI and business logic

## Example

src/
├── app/
├── components/
│ ├── ui/
│ ├── common/
│ └── features/
├── hooks/
├── services/
├── stores/
├── utils/
├── types/
├── constants/
├── lib/
└── styles/
